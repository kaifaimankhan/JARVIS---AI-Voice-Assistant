# JARVIS - AI Voice Assistant

A Python-based AI Voice Assistant that listens for a wake word ("Jarvis") and executes voice commands using Speech Recognition, Artificial Intelligence, and Text-to-Speech technologies.

## Features

* Wake word detection
* Voice command recognition
* AI-powered conversations
* Open Google, YouTube, Facebook, and LinkedIn
* Play songs directly on YouTube
* Read the latest news headlines
* Natural voice responses
* Speech-to-Text and Text-to-Speech integration
* Error handling for internet and speech recognition issues

## Technologies Used

* Python
* SpeechRecognition
* OpenAI API
* gTTS

## Installation

Clone the repository:

```
git clone https://github.com/yourusername/jarvis-ai.git
```

Install dependencies:

```
pip install -r requirements.txt
```

## Required API Keys

Create a `.env` file:

```
OPENAI_API_KEY=your_api_key
NEWS_API_KEY=your_news_api_key
```

## How to Run

```
python main.py
```

Say:

```
Jarvis
```

Then give commands like:

```
Open Google
Play Believer
Open YouTube
Tell me the news
Who is Alan Turing?
```
